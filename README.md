# 📐 Matemáticas ESRN-11

Aplicación web progresiva (PWA) para el aprendizaje interactivo de matemáticas.

## ✨ Características

- 🎨 **Diseño Moderno** - UI/UX profesional con glassmorphism
- 📱 **100% Responsive** - Funciona en todos los dispositivos
- 🌙 **Modo Oscuro** - Adaptación automática al tema del sistema
- 📲 **Instalable** - Tarjeta de instalación nativa
- 📶 **Offline** - Funciona sin conexión a internet
- ⚡ **Rápida** - Carga instantánea con service worker
- ♿ **Accesible** - Cumple estándares WCAG 2.1 AA

## 🚀 Instalación

### En GitHub Pages
1. Ve a **Settings** → **Pages**
2. Selecciona la rama `main`
3. Tu app estará en `https://27698-sis.github.io/matematica-esrn-11/`

### Localmente
```bash
git clone https://github.com/27698-sis/matematica-esrn-11.git
